# SciBoost 科技研发资金募集网站

一个专业的科技研发资金募集平台，连接创新科技与资金支持。

- 清爽简洁的界面布局

✅ **完整功能模块**

## 使用方法

### 方式1：直接打开
1. 下载所有文件到同一文件夹
2. 双击 `index.html` 文件
3. 在浏览器中查看网站

### 方式2：本地服务器（推荐）
使用Python启动本地服务器：
```bash
python -m http.server 8000
```
或使用Node.js：
```bash
npx serve .
```
然后在浏览器访问：http://localhost:8000

### 方式3：部署到生产环境
将文件上传到任何静态网站托管服务：
- GitHub Pages
- Netlify
- Vercel
- 阿里云OSS
- 腾讯云COS

## 自定义内容

### 修改项目信息
在 `index.html` 中找到 `.project-card` 区域，修改以下内容：
- 项目名称（`<h3 class="project-title">`）
- 项目描述（`<p class="project-description">`）
- 目标金额、已筹金额、完成度
- 项目图片URL

### 修改联系方式
在 `index.html` 中找到 `.contact-info` 区域，更新：
- 邮箱地址
- 电话号码
- 公司地址
- 社交媒体链接

### 修改品牌信息
- 网站标题：修改 `<title>` 标签
- Logo文本：修改 `.logo-text`
- 品牌颜色：在 `styles.css` 中修改CSS变量

### 修改银行账户信息
在 `donation.html` 中找到 `bankAccounts` JavaScript对象，更新：
- 账户名称、账号、银行代码等信息
- 不同国家的银行详情
- 备注说明和提示文字

### 更新捐款者名录
在 `donors.html` 中找到捐款者展示区域，更新：
- 捐款者姓名（支持匿名显示）
- 捐款金额和日期
- 捐款等级（钻石、白金、黄金等）
- 捐款留言（可选）

## 支付页面功能

### 多币种支持
支付页面 (`donation.html`) 支持以下国家和地区的银行转账：
- 🇬🇧 英国 (GBP)
- 🇦🇺 澳大利亚 (AUD)
- 🇨🇦 加拿大 (CAD)
- 🇨🇳 中国 (CNY)
- 🇪🇺 欧元区 (EUR)
- 🇭🇺 匈牙利 (HUF)
- 🇳🇿 新西兰 (NZD)
- 🇸🇬 新加坡 (SGD)
- 🇹🇷 土耳其 (TRY)
- 🇺🇸 美国 (USD)

### 核心功能
- **国旗图标选择**：点击国家旗帜图标选择支付方式
- **居中弹窗显示**：页面居中弹出账户详情，高度自适应不遮挡导航栏
- **一键复制**：
  - 复制全部账户信息
  - 复制单个字段（账号、IBAN、SWIFT等）
- **复制反馈**：显示成功提示消息
- **使用说明**：提供详细的转账备注和联系信息

## 捐款者名录功能

### 荣誉墙设计
捐款者名录页面 (`donors.html`) 采用荣誉墙设计风格：
- **捐款等级展示**：钻石、白金、黄金、白银、青铜、支持者六个等级
- **捐款统计**：显示总捐款人次、累计金额、支持项目数
- **隐私保护**：支持匿名显示，只显示姓氏

### 核心功能
- **等级标识**：每个等级有独特的颜色和图标标识
- **筛选功能**：按捐款等级筛选（全部、钻石级、白金级等）
- **排序功能**：
  - 按捐款日期排序（最新/最早）
  - 按捐款金额排序（高到低/低到高）
- **加载更多**：分页加载，优化性能
- **捐款留言**：展示捐款者的留言和祝福
- **响应式卡片**：适配不同屏幕尺寸

### 页面结构
- **头部统计**：显示总体捐款数据
- **等级说明**：展示各等级的捐款金额范围和当前人数
- **筛选区域**：提供筛选和排序控件
- **荣誉墙**：以卡片形式展示捐款者信息
- **CTA区域**：引导更多用户参与捐款
