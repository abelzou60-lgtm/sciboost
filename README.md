一个专业的科技研发资金募集平台，连接创新科技与资金支持。

- 蓝色+白色主色调，科技感十足

- 清爽简洁的界面布局

✅ **完整功能模块**
- Hero 展示区域
- 科研项目展示

- 科技研发介绍

✅ **丰富的交互效果**
- 平滑滚动导航

## 使用方法

### 方式1：直接打开
1. 下载所有文件到同一文件夹
2. 双击 `index.html` 文件
3. 在浏览器中查看网站

### 方式2：本地服务器（推荐）
使用Python启动本地服务器：
```bash
python -m http.server 8000
```
或使用Node.js：
```bash
npx serve .
```
然后在浏览器访问：http://localhost:8000

### 方式3：部署到生产环境
将文件上传到任何静态网站托管服务：
- GitHub Pages
- Netlify
- Vercel
- 阿里云OSS
- 腾讯云COS

## 自定义内容

### 修改项目信息
在 `index.html` 中找到 `.project-card` 区域，修改以下内容：
- 项目名称（`<h3 class="project-title">`）
- 项目描述（`<p class="project-description">`）
- 目标金额、已筹金额、完成度
- 项目图片URL

### 修改联系方式
在 `index.html` 中找到 `.contact-info` 区域，更新：
- 邮箱地址
- 电话号码
- 公司地址
- 社交媒体链接

### 修改品牌信息
- 网站标题：修改 `<title>` 标签
- Logo文本：修改 `.logo-text`
- 品牌颜色：在 `styles.css` 中修改CSS变量
